# 7장 표본크기가 클 때

## 일관성 확인


```python
import numpy as np
import statsmodels.api as sm

n = 1000000
np.random.seed(13579)
x = np.random.normal(size=n)
u = np.random.normal(size=n)
y = [1+0.7*xi+ui for xi,ui in zip(x,u)]
sm.OLS(y, sm.add_constant(x)).fit().params
```




    array([0.9997532 , 0.70088215])
