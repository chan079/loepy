## 예제 12.4 지수분포 가정하의 최우추정(예제 12.1 계속)


```python
import numpy as np
from scipy import optimize

def logL(a):
    return -(10*np.log(a)-7.7*a) # for minimization

optimize.minimize(logL, 2)
```
